Metcalfe's Law Network Animator

This interactive tool visualizes networks based on Metcalfe's Law, the principle that the value of a network is proportional to the square of its users. Adjust parameters to see how network structure and aesthetics change. Create and download animations of network growth. Read the blog post for more detail. By Jeff Veen.
